# خمسات سنايبر — Khamsat Sniper

إضافة كروم ذكية تراقب طلبات خمسات وتولّد عروضاً احترافية بالذكاء الاصطناعي تلقائياً.

> A Chrome extension that monitors Khamsat service requests and auto-generates professional AI-powered proposals.

---

## التثبيت / Installation

### الطريقة 1 — من GitHub Releases (موصى بها)

1. اذهب إلى صفحة [**Releases**](../../releases/latest)
2. حمّل ملف **`khamsat-sniper-vX.X.X.zip`** من قسم Assets
3. **افك ضغط الملف** إلى مجلد على جهازك
4. افتح المتصفح وانتقل إلى `chrome://extensions`
5. فعّل **Developer mode** من الزاوية العلوية اليمنى
6. اضغط **Load unpacked** واختر المجلد الذي فككت إليه الضغط
7. ستظهر أيقونة الإضافة في شريط الأدوات ✓

> **تنبيه:** اختر المجلد نفسه — لا تختر ملف `.zip` مباشرةً

---

### Method 1 — From GitHub Releases (Recommended)

1. Go to the [**Releases**](../../releases/latest) page
2. Download **`khamsat-sniper-vX.X.X.zip`** from the Assets section
3. **Extract the ZIP** to any folder on your computer
4. Open your browser and navigate to `chrome://extensions`
5. Enable **Developer mode** using the toggle in the top-right corner
6. Click **Load unpacked** and select the folder you extracted
7. The extension icon will appear in your toolbar ✓

> **Important:** Select the extracted *folder* — do not select the `.zip` file directly

---

## الإعداد / Setup

1. اضغط على أيقونة الإضافة في شريط الأدوات
2. اضغط **الإعدادات**
3. اختر مزوّد الذكاء الاصطناعي (Groq مجاني وسريع)
4. أدخل مفتاح API الخاص بك
5. اختر مجالات عملك ومهاراتك
6. اضغط **حفظ الإعدادات**

---

1. Click the extension icon in the toolbar
2. Click **Settings**
3. Choose an AI provider (Groq is free and fast)
4. Enter your API key
5. Select your fields of work and skills
6. Click **Save Settings**

## كيفية الاستخدام / How to Use

1. افتح [صفحة طلبات خمسات](https://khamsat.com/community/requests)
2. اضغط **ابدأ المراجعة** من الزر العائم في أسفل الشاشة
3. الإضافة ستحلل كل طلب وتعرض نسبة التطابق مع مهاراتك
4. على الطلبات المطابقة، اضغط **رد** لتوليد عرض احترافي
5. راجع النص وعدّله، ثم انسخه أو أدرجه مباشرةً

---

1. Open the [Khamsat requests page](https://khamsat.com/community/requests)
2. Click **Start Review** from the floating button at the bottom
3. The extension analyzes each request and shows a match score
4. On matching requests, click **Reply** to generate a professional proposal
5. Review and edit the text, then copy or insert it directly

---

## الخصوصية / Privacy

- جميع مفاتيح API محفوظة **محلياً** على جهازك فقط
- لا يتم إرسال أي بيانات لخوادم خارجية
- الاتصالات الوحيدة هي مع موقع خمسات ومزوّد AI الذي تختاره

---

- All API keys are stored **locally** on your device only
- No data is sent to any external server
- The only connections are to khamsat.com and your chosen AI provider
