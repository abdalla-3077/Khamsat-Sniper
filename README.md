# 🎯 خمسات سنايبر - إضافة كروم

إضافة كروم ذكية لمراقبة طلبات الخدمات على منصة خمسات وتوليد عروض مخصصة باستخدام الذكاء الاصطناعي.

## ✨ المميزات

- مراقبة صفحة طلبات خمسات لاكتشاف الطلبات المطابقة
- توليد عروض احترافية بالذكاء الاصطناعي
- دعم مزوّدين متعددين: OpenAI، Groq، Mistral
- واجهة عربية بالكامل مع دعم RTL
- تصميم داكن عصري
- لا ينشر أي رد تلقائياً - أنت من يقرر النشر

## 📦 التثبيت

1. حمّل الملفات إلى مجلد واحد
2. افتح كروم وانتقل إلى `chrome://extensions`
3. فعّل "وضع المطور" (Developer mode) في الزاوية اليمنى العليا
4. اضغط "Load unpacked" (تحميل فك الضغط)
5. اختر مجلد الإضافة
6. ستظهر أيقونة الإضافة في شريط الأدوات

## ⚙️ الإعدادات

### إعدادات المراقبة
- **تفعيل المراقبة**: شغّل أو أوقف الفحص التلقائي
- **فترة التحقق**: الوقت بين كل فحص (بالدقائق)
- **الكلمات المفتاحية**: أدخل الكلمات المطابقة (كل كلمة في سطر)

### إعدادات الذكاء الاصطناعي
1. اختر المزوّد المناسب (Groq أسرع وأرخص)
2. أدخل مفتاح API الخاص بالمزوّد
3. اختر النموذج المناسب
4. أدخل معلوماتك ونمط ردودك في قسم "شخصيتي"

## 🔑 الحصول على مفاتيح API

### OpenAI
1. انتقل إلى [platform.openai.com](https://platform.openai.com)
2. سجّل الدخول أو أنشئ حساباً جديداً
3. اذهب إلى API Keys
4. اضغط "Create new secret key"
5. انسخ المفتاح

### Groq (موصى به)
1. انتقل إلى [console.groq.com](https://console.groq.com)
2. سجّل الدخول
3. اذهب إلى API Keys
4. اضغط "Create API Key"
5. انسخ المفتاح

### Mistral
1. انتقل إلى [console.mistral.ai](https://console.mistral.ai)
2. سجّل الدخول
3. اذهب إلى API Keys
4. اضغط "Create new key"
5. انسخ المفتاح

## 📖 كيفية الاستخدام

1. **الإعداد**: افتح الإعدادات وأدخل مفاتيح API والكلمات المفتاحية
2. **المراقبة**: افتح صفحة طلبات خمسات واضغط "تحقق الآن"
3. **المطابقة**: الطلبات المطابقة ستظهر بعلامة "مطابق" وزر "توليد عرض"
4. **التوليد**: اضغط الزر لتوليد عرض مخصص
5. **المراجعة**: راجع النص وعدّله إذا لزم الأمر
6. **الإدراج**: اضغط "استخدم هذا الرد" لإدراج النص في حقل الرد
7. **النشر**: اضغط زر النشر يدوياً على الموقع

## 🔒 الخصوصية

- جميع المفاتيح محفوظة محلياً فقط على جهازك
- لا يتم إرسال أي بيانات لخوادم خارجية
- فقط اتصالات API مباشرة للمزوّد المختار

## 🐛 حل المشاكل

- **لم تظهر الطلبات المطابقة**: تأكد من إدخال كلمات مفتاحية صحيحة
- **خطأ في API**: تأكد من صحة المفتاح والمزوّد المختار
- **لم يتم إدراج الرد**: حاول تحديث الصفحة

## 📝 ملاحظات

- الإضافة لا تنشر أي رد تلقائياً
- أنت المسؤول عن مراجعة ونشر العروض
- تأكد من إعداد حساب مناسب على المزوّد المختار
